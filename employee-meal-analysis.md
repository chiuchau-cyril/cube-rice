## 四、與現有外包的比較

**請填入目前外包費用後對比：**

| | 外包現況 | 自營（洪達方案） |
|---|---------|-----------------|
| 每人每餐成本 | $___  | **~$101**（$111K ÷ 50人 ÷ 22天）|
| 月總費用 | $___  | **$111,000** |
| 品質掌控 | ❌ 差（你已說外包做得差）| ✅ 自己掌控 |
| 菜單彈性 | ❌ 看外包商 | ✅ 可自訂 |
| 初始投資 | $0 | **$920,000** |
| 回本條件 | - | 每月省下 $___（外包費 - $111K）|

### 回本試算

```
假設目前外包費用 = $130/人/餐（含利潤）
月外包費用 = 50人 × $130 × 22天 = $143,000
月節省 = $143,000 - $111,000 = $32,000
回本期 = $920,000 ÷ $32,000 = 28.8 個月 ≈ 2.4 年

假設目前外包費用 = $100/人/餐
月外包費用 = 50人 × $100 × 22天 = $110,000
月節省 = $110,000 - $111,000 = -$1,000
→ 成本打平，但品質大幅提升
```

## 五、隱性價值（不計入 ROI 但很重要）

| 價值 | 說明 |
|------|------|
| 🧪 **設備測試場域** | 每天實際操作洪達設備，累積經驗與數據 |
| 📊 **食材標準化實驗** | 可以開始測試 Cube Food 方塊食材的可行性 |
| 🎥 **影片素材** | 拍攝實際烹調過程，未來洽談合作的展示材料 |
| 🍳 **食譜開發** | 在洪達設備上開發炒飯程式，累積配方參數 |
| 👨‍🍳 **人員培訓** | 專員成為設備操作專家，未來可帶領團隊 |
| 😊 **員工滿意度** | 吃得更好 → 團隊士氣提升 |

## 六、建議決策流程

```
Step 1: 確認員工人數與目前外包費用 → 計算實際 ROI
Step 2: 確認廚房空間與電力條件
Step 3: 決定是否購入洪達設備
Step 4: 招聘廚務專員（或先內部調配）
Step 5: 向洪達下單，同時進行廚房裝修
Step 6: 設備到場 → 2 週測試期
Step 7: 正式供餐
```

### 時程估計

| 階段 | 時間 |
|------|------|
| 決策 + 下單 | 1-2 週 |
| 廚房裝修 | 2-3 週 |
| 設備交付 + 安裝 | 2-4 週（待確認洪達交期） |
| 試運行 | 2 週 |
| **預計啟用** | **約 2 個月後** |
