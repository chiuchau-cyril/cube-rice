## 二、供餐模式設計

### 每日供餐流程

```
08:00-09:00  專員到班，食材驗收入庫
09:00-10:30  備料：洗切、分裝至洪達給料盒、煮飯、煮湯
10:30-11:30  烹調：洪達炒菜機連續出菜（每道 5-8 分鐘）
11:30-12:30  供餐時段（自助取餐）
12:30-13:30  收餐、清洗（碗盤機）、設備清潔、廚餘處理
13:30-14:00  廚房整理、隔日食材盤點/採購下單
```

### 每日菜單結構

| 項目 | 內容 | 設備 |
|------|------|------|
| 主菜 1 | 肉類熱炒（如：宮保雞丁、黑胡椒牛柳）| 洪達炒菜機 |
| 主菜 2 | 海鮮/蛋類熱炒（如：蝦仁炒蛋、魚香豆腐）| 洪達炒菜機 |
| 配菜 | 蔬菜熱炒（如：炒高麗菜、炒四季豆）| 洪達炒菜機 |
| 白飯 | 60 人份（約 12kg 白米） | 蒸飯機 |
| 湯品 | 每日一湯（如：紫菜蛋花湯、味噌湯）| 電磁爐 + 大湯鍋 |

### 菜色輪替表（2 週循環範例）

| 週 | 一 | 二 | 三 | 四 | 五 |
|---|---|---|---|---|---|
| **W1 主菜** | 宮保雞丁 | 黑胡椒牛柳 | 蝦仁炒蛋 | 糖醋排骨 | 三杯雞 |
| **W1 配菜** | 炒高麗菜 | 炒四季豆 | 蒜炒空心菜 | 番茄炒蛋 | 炒花椰菜 |
| **W2 主菜** | 沙茶牛肉 | 魚香茄子 | 蔥爆豬肉 | 宮保蝦仁 | 客家小炒 |
| **W2 配菜** | 炒青江菜 | 蒜炒大陸妹 | 螞蟻上樹 | 炒木耳 | 麻婆豆腐 |

每日 2 主菜 + 1 配菜 + 白飯 + 湯 = **完整午餐**

### 產能分析

| 參數 | 數值 |
|------|------|
| 洪達單道菜烹調時間 | ~5-8 分鐘（含清洗） |
| 每道菜份量 | 預估 10-15 人份/鍋 |
| 60 人所需鍋數 | 主菜 1（4-6 鍋）+ 主菜 2（4-6 鍋）+ 配菜（4-6 鍋）= **12-18 鍋** |
| 所需時間 | 18 鍋 × 8 分鐘 = **~150 分鐘（2.5 小時）** |
| 開始烹調 | 09:00 |
| 全部完成 | 11:30 ✅ 供餐前完成 |

> 一台洪達炒菜機足夠應付 60 人午餐，不需要買兩台。

## 六、廚務專員招聘

### 職缺資訊

| 項目 | 內容 |
|------|------|
| 職稱 | 廚務專員 / 員工餐廳管理員 |
| 工作時間 | 週一至週五 08:00-14:00（6 小時） |
| 薪資 | $35,000-42,000/月（含勞健保） |
| 學歷 | 不拘 |
| 證照 | 丙級中餐烹調技術士（優先）或食品衛生相關 |

### 工作內容

1. **食材管理**（08:00-09:00）
   - 驗收每日/隔日食材配送
   - 食材入庫、冷藏/冷凍管理
   - 盤點庫存、下單採購

2. **備料**（09:00-10:30）
   - 食材洗切、醃製
   - 依菜單將食材分裝至洪達給料盒（4 盒）
   - 設定洪達炒菜機程式（選擇菜單）
   - 煮飯、煮湯

3. **烹調**（10:30-11:30）
   - 操作洪達炒菜機出菜
   - 每道菜完成後裝盤至保溫台
   - 監控設備運作

4. **供餐與清潔**（11:30-14:00）
   - 供餐時段管理
   - 碗盤清洗機操作
   - 廚房清潔、設備保養
   - 廚餘處理
   - 隔日菜單規劃

### 招聘管道建議

- 104 人力銀行
- 1111 人力銀行
- 在地社區/里辦公處公告
- 餐飲科系學校（實習合作）
