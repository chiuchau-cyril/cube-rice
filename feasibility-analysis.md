## 🎯 關鍵問題：94 單/天 達得到嗎？

### 拆解日單量

假設 24H 營運，訂單分佈：

| 時段 | 時長 | 佔比 | 94單拆解 |
|------|------|------|----------|
| 早餐 (6-10) | 4hr | 5% | 5 單 |
| 午餐 (10-14) | 4hr | 35% | 33 單 |
| 下午 (14-17) | 3hr | 5% | 5 單 |
| 晚餐 (17-21) | 4hr | 40% | 38 單 |
| 宵夜 (21-02) | 5hr | 12% | 11 單 |
| 深夜 (02-06) | 4hr | 3% | 3 單 |

**午餐尖峰：33單 ÷ 4小時 = 8.3單/小時 = 每 7.2 分鐘 1 單**

### 產能檢驗

| 參數 | 數值 |
|------|------|
| 炒鍋數量 | 3 台 |
| 單鍋出餐時間 | 4 分鐘 |
| 理論產能 | 3 × 15 = 45 單/小時 |
| 尖峰需求 | 8.3 單/小時 |
| 產能利用率 | 18% ✅ 綽綽有餘 |

**結論：產能不是問題，訂單量才是。**

## ⚖️ SWOT 分析

### 優勢 Strengths
- ✅ 24H 營運，覆蓋深夜市場（競爭少）
- ✅ 品質穩定，不受廚師狀態影響
- ✅ 人力成本極低
- ✅ 標準化易複製

### 劣勢 Weaknesses
- ❌ 高前期投資（$270萬）
- ❌ 菜單受限（只能做標準化品項）
- ❌ 無法客製化（少辣、不要蔥等）
- ❌ 品牌認知度從零開始
- ❌ 設備故障 = 停業

### 機會 Opportunities
- 📈 外送市場持續成長
- 📈 人力成本上漲，自動化優勢擴大
- 📈 工業區午餐剛需穩定
- 📈 深夜市場尚未飽和

### 威脅 Threats
- ⚠️ 平台抽成可能調漲
- ⚠️ 傳統店家降價競爭
- ⚠️ 消費者對「機器做的」接受度
- ⚠️ 設備商倒閉/維修困難

## 🎯 結論：可行嗎？

### 數據判決

| 評估面向 | 結論 |
|----------|------|
| 技術可行性 | ✅ 可行（設備存在且成熟）|
| 財務可行性 | 🟡 有條件可行 |
| 市場可行性 | 🟡 需驗證 |
| 營運可行性 | 🟡 有風險 |

### 核心結論

```
📌 這個商業模式「不是不可行」，但成敗完全取決於一件事：

   → 你能不能穩定獲得 100+ 單/天？

   如果可以 → 這是一門好生意
   如果不行 → 就是燒錢
```

### 建議策略

#### 🟢 應該做
1. **先驗證訂單量**：租一個現有廚房，用傳統方式試賣 2-3 個月
2. **鎖定利基市場**：專攻深夜時段，競爭最少
3. **控制初期投資**：先用 2 台炒鍋，驗證後再擴

#### 🔴 不應該做
1. 一開始就投 $270 萬全套設備
2. 相信「蓋好就會有訂單」
3. 忽略品牌和行銷

## 📎 附錄：計算明細

### A. 貢獻利潤推導
```
客單價                    $100
- 平台抽成 (30%)          -$30
= 實收                     $70
- 食材成本 (30%)          -$30
- 包材                     -$8
= 貢獻利潤                 $32
```

### B. 損益平衡點
```
固定成本 / 貢獻利潤 / 30天
= $90,000 / $32 / 30
= 93.75 ≈ 94 單/天
```

### C. 回本期計算
```
CAPEX / 年淨利
= $2,700,000 / ($54,000 × 12)
= $2,700,000 / $648,000
= 4.17 年（日單量 150 時）
```
